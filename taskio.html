Taskio

Idea

To push myself to the limits, I decided to learn a Javascript framework. I didn't had the chance to learn a framework at school, so I needed to learn it myself. After some research, Vue and React came on top. These frameworks are reliable and extremely populair. Since my internship company used Vue, I decided to learn Vue first. A simple To-Do application was the goal.

Process

First of all, I made a design in Adobe XD. A simple to-do list. With the this design I had a goal in mind. So I started with reading documentation on the Vue.js website. It was well documented and I created a Vue 3 application in seconds. After the installation I had a simple webpage with a few elements.

I deleted all the elements and started from scratch. Then I first made the entire tasks page. So the title with colored balls and placeholder for the list items. After that I did some research on building reactive elements with Vue 3. I came across the composition API. This was a really good way to build a To-Do application.

So the last steps were to make the list interactive, a local storage functionality and a loading screen. I made these functionalities after I saw some explanatory videos and websites.

What did I learn?

I learned about Javascript frameworks and in special Vue.js

These are the skills that I have learned in this project:



- Javascript frameworks

- Vue.js
